# JY Portfolio

프론트엔드 개발자 허정연의 포트폴리오 사이트입니다.

🔗 **[developer-pino.info](https://developer-pino.info)**

<br>

## 소개

디자인과 코드를 잇는 프론트엔드 개발자로, 감각적인 UI와 견고한 구조를 함께 추구합니다.
실무 프로젝트 소개, 기술 블로그, 오픈소스 활동, PT 자료까지 한 곳에 담았습니다.

<br>

## 주요 기능

- **다크모드 / 라이트모드** 전환 (시스템 설정 연동)
- **한국어 / 영어** 전환
- Hero 섹션 인터랙티브 드로잉 캔버스
- 챗봇 — 자주 묻는 질문에 즉시 응답
- 프로젝트 상세 페이지, 블로그, 오픈소스, PT 자료 페이지
- Supabase 기반 CMS — 관리자 페이지에서 콘텐츠 직접 관리

<br>

## 기술 스택

| 분류 | 사용 기술 |
|------|-----------|
| Frontend | React 18, TypeScript, Vite |
| Styling | styled-components, Tailwind CSS |
| Routing | React Router v6 |
| Animation | Motion (Framer Motion) |
| Backend / DB | Supabase (PostgreSQL, Storage, Auth) |
| 배포 | Vercel |
